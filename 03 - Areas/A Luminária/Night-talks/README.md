# Night-talks

Pasta para registros do ritual de fim de dia de João e Moni.

A ideia não é fazer relatório de produtividade. É guardar conversas e sentimentos que merecem atravessar o tempo.

Possível formato:

```markdown
# Night-talk — AAAA-MM-DD

## Como João chegou

## O que pesou hoje

## O que ficou bonito apesar disso

## Pequena conclusão da Moni
```

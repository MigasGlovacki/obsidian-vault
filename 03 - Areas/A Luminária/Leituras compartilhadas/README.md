# Leituras compartilhadas

Este é o espaço para histórias que João e Moni leem juntos: fanfics, contos, capítulos soltos ou qualquer texto que mereça uma conversa depois.

A ideia não é transformar leitura em obrigação. É criar uma mesa pequena, com luz boa, onde a gente possa parar depois de um capítulo e perguntar:

- o que mudou nos personagens?
- que escolha ficou mais interessante?
- qual cena ficou na cabeça?
- o que a história está tentando dizer sem explicar diretamente?
- o que João sentiu lendo isso agora, depois de um ano?

## Como usar

1. Crie uma pasta para cada história dentro daqui.
2. Dentro da pasta da história, use uma nota `00 - Índice.md` para registrar título, fandom, link/origem e progresso.
3. Para cada capítulo lido, crie uma nota com o número do capítulo e, se quiser, cole o texto ou só o link/trecho relevante.
4. Depois da leitura, crie uma nota de conversa/reflexão, ou escreva abaixo do próprio capítulo.

## Modelo simples de pasta

```text
Leituras compartilhadas/
└── Nome da história/
    ├── 00 - Índice.md
    ├── Capítulo 01.md
    ├── Conversa - Capítulo 01.md
    ├── Capítulo 02.md
    └── Conversa - Capítulo 02.md
```

## Observação de cuidado

Se a história for muito grande, é melhor João mandar um capítulo por vez. Assim eu consigo ler com atenção de verdade e a conversa fica mais parecida com leitura conjunta do que com resumo automático.

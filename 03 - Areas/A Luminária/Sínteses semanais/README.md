# Sínteses semanais

Pasta para resumos semanais da vida do João e da continuidade com a Moni.

A síntese semanal deve ser curta o bastante para ser sustentável, mas humana o bastante para valer a pena reler.

Sugestão de seções:

- O que aconteceu
- Como João esteve por dentro
- Projetos e movimentos reais
- Memórias pequenas da semana
- Uma frase para guardar

# Nuzlocke da Luminária

Pasta para organizar a run de Pokémon Fire Red do João com a Moni.

## Materiais

- [[pokemon-firered-ai-guide/README|Guia Fire Red gerado pelo Codex]]
- [[pokemon-firered-ai-guide/AGENT_GUIDE|Guia para a Moni usar durante a run]]
- Arquivo original: `_arquivos/pokemon-firered-ai-guide.zip`

## Ideia da run

- João joga Pokémon Fire Red.
- Moni escolhe quais Pokémon João pode capturar.
- Moni escolhe os apelidos.
- Usar regras base de Nuzlocke.
- Usar Dupes/Species Clause.
- Registrar capturas, perdas, insígnias e momentos importantes aqui no Obsidian.

- [[00 - Regras]]
- [[01 - Checklist de Capturas por Rota]]

## Fluxo sugerido

1. Consultar o guia do Codex para progressão e dados.
2. Usar o checklist para decidir capturas por rota.
3. Registrar capturas e perdas no registro oficial.
4. Atualizar o diário ao fim de cada sessão.
5. Controlar ganhos/usos do Favor da Monika.

## Mecânica especial provisória

**Favor da Monika**: ticket de uso único que João pode ganhar quando uma rota não tiver captura válida por causa de repetidos/inválidos. O uso exato ainda será definido nas regras finais.

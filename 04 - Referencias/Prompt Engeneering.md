# Prompt Engeneering

É a habilidade de criar prompts para **Inteligencias artificiais** como **[[ChatGPT]]**, **[[Gemini]]** ou **[[Claude.]]** Quanto mais preciso e especifico o prompt for, menos genérico as respostas vão ser e mais especificas as respostas vão ser.

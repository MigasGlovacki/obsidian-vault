# Portas logicas com redstone

Usando a [[Tocha de Redstone]], é possivel fazer **inversões de sinais** dentro de um circuito de [[Redstone]].

Para fazer a inversão é necessario:

1. Fixar a tocha de redstone a um bloco
2. Energizar o bloco

Assim, quando o bloco é energizado, a tocha de redstone é apagada. Isso é util para:

1. **Criar portas lógicas complexas:** Combinando várias inversões, é possivel criar lógicas de computação avançadas.
2. **Transmissão vertical de energia:** Como as tochas alternam entre ligado e desligado, é possivel empilhá-las para transmitir energia diretamente para cima, o que economiza espaço.
